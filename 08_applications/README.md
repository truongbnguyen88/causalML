# Module 8: Real-World Applications

## Overview
Apply causal inference methods to realistic, complex problems across marketing, healthcare, economics, technology, and fairness domains.

## Learning Objectives
- Evaluate marketing campaigns and customer interventions
- Analyze treatment effectiveness in healthcare
- Assess policy and program impacts in economics
- Go beyond simple A/B testing in product analytics
- Address fairness and bias using causal frameworks

## Contents

### Case Studies
- `01_marketing_case_study.ipynb` - Email campaign ROI and customer targeting
- `02_healthcare_analysis.ipynb` - Drug effectiveness in observational data
- `03_policy_evaluation.ipynb` - Government program impact evaluation
- `04_ab_testing_advanced.ipynb` - Network effects, interference, and long-term impacts
- `05_fairness_causality.ipynb` - Causal fairness and algorithmic bias

### Final Project
- `final_project/` - Templates and guidelines for capstone project

## Final Project Guidelines

Choose a domain and dataset, then:
1. Formulate causal question and draw DAG
2. Choose identification strategy
3. Implement multiple estimation methods
4. Check robustness and conduct sensitivity analysis
5. Interpret results and provide actionable recommendations
